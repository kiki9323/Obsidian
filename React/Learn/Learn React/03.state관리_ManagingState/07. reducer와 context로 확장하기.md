#### flow summary
* Reducer: Reducer를 사용하면 컴포넌트의 state 업데이트 로직을 통합할 수 있다. 
* Context: Context를 사용하면 다른 컴포넌트들에 정보를 전달할 수 있다. 
* Reducer + Context:  Reducer와 Context를 함께 사용하면 복잡한 화면의 state를 관리할 수 있다.
